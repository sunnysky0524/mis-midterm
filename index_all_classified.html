✏️ 必考填充題

考填充 這 4 題要背順序與完整答案。

1. 電子商務依交易對象的不同，可分為哪四種類型？
  1. B2B
  2. B2C
  3. C2C
  4. C2B
B2B、B2C、C2C、C2B
2. 客戶關係管理分為哪三個階段？需按照步驟順序回答。
  1. 獲取
  2. 增進
  3. 維持
獲取 → 增進 → 維持
3. 客戶關係管理包含哪四個流程？需按照步驟順序回答。
  1. 知識發現
  2. 行銷規劃
  3. 客戶互動
  4. 分析修正
知識發現 → 行銷規劃 → 客戶互動 → 分析修正
4. 網路社群提供多元且豐富的功能，具有哪三個特性？
  1. 虛擬性
  2. 開放性
  3. 獨特性
虛擬性、開放性、獨特性

⚡ 考前速讀

電子商務四類型

B2B → B2C → C2C → C2B
  • B2B：企業對企業
  • B2C：企業對消費者
  • C2C：消費者對消費者
  • C2B：消費者對企業

CRM 三階段

獲取 → 增進 → 維持
  • 獲取新客戶
  • 增進現有客戶價值
  • 維持具有價值客戶

CRM 四流程

知識 → 行銷 → 客戶 → 分析
  • 知識發現
  • 行銷規劃
  • 客戶互動
  • 分析修正

網路社群三特性

虛擬性 → 開放性 → 獨特性
  • 虛擬性
  • 開放性
  • 獨特性
